请求参数
参数名位置类型必填说明
AuthorizationHeaderstring是Bearer {access_token}
pageQueryint否页码，默认 1
pageSizeQueryint否每页数量，默认 20
startDateQuerystring否开始日期 YYYY-MM-DD

快速开始
1
安装 SDK
选择你的开发语言，安装对应 SDK 包
2
获取 API Key
在开发者管理页面创建应用，获取 App ID 和 API Key
3
初始化客户端
用 App ID 和 API Key 初始化 SDK 客户端
4
调用 API
调用日程、任务、通知等 API，参考开发文档
代码示例
# Python 示例：获取日程列表 from rishiqing import Client client = Client("app_001", "sk_live_xxxx7f8g") calendars = client.calendar.list(page=1, pageSize=20) for cal in calendars.list: print(f"{cal.title} - {cal.startTime}")

4. 开发者：API Key管理，创建/禁用开发者账号
5. 开发文档：OpenAPI/Swagger文档，开发者自助查阅
